About teppy.rocks

Here's the part specifically intended for curious individuals. This contains a summary of the site's features, as well as those I intend to add in the near future. (I only wanted to have this online ASAP. All forum functionalities are not yet implemented but the site, as a "personal blog site", is functional.)

Created with separated Authed and Un-authed views

Perhaps the feature I really worked myself off for was this one. I literally spent days separating what a logged-in user versus an anonymous user can see and do...previously. The site still has that feature but "guests" can freely browse the contents. Log-in-Required functionalities are only for those who wish to contribute to the blogs and shared stuffs.

Social Media-like

Since I originally intended this site to be a forum, user accounts created are intended to have to display "distinct user info" (I will not call it "personal info" since they can all be dummy information just for users to have something to display). It'll be up to the users to add personal information should they determine that it will help people reach-out to them if needed.

Cross-Site Request Forgery (CSRF) protection

Another reason I stick with the Django framework was because I was convinced of its built-in security features.

Since this site features allowing users other than the admin to be writers/bloggers, as well, we need a better way of protecting unsuspecting victims of online frauds using HTML/SQL injections. I am convinced that Django's security features are what we need to minimize the risks. Going a little technical mentioning it here, you can check this out for more information. The first paragraph will give you a summary of what you need to know.
Please note that Still...and Always, in the online world: Vigilance is a must. Always see
